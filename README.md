### 💡 About Me

I build **AI systems that go from research to production**, not just notebooks that live and die on a laptop. My work spans **RAG, multi-agent orchestration, local/quantized LLM inference, and real-time multimodal pipelines**.

- 🔭 Currently running **local, zero-API AI systems** — 4-bit quantized LLMs on consumer GPUs, because inference costs shouldn't gatekeep good products
- 🧠 Deep in **LangGraph multi-agent pipelines**, **RAG architectures**, and **fine-tuning with LoRA**
- 🎙️ Recently shipped a **full-duplex Arabic voice tutor** (STT → LLM → RAG → TTS, all local, streaming)
- 🏗️ I care about **clean architecture and SOLID principles** as much as model accuracy — AI systems still need to be *engineered*
- 🌱 Studying **AI at Helwan University's Faculty of AI**, Cairo
- 💼 Freelance AI/GenAI Engineer — delivered production systems for clients in **travel, healthcare, education, and SaaS**

### 🛠️ Featured Builds

| Project | What it does | Stack |
|---|---|---|
| **[Multi-Agent Research Automation](#)** | 5-agent pipeline (Planner → Scraper → Analyzer → Writer → Critic) turns a query into a sourced research report in under 3 minutes — 100% local, with hallucination-auditing critic loops | LangGraph, ChromaDB, FastAPI, Ollama (Llama 3.1 8B) |
| **Real-Time Arabic Voice Tutor** | Full-duplex speech-to-speech tutoring: Whisper STT → quantized 13B LLM → RAG → TTS with a live avatar UI | Whisper, Ollama, RAG, React, Tavus |
| **Transformer LLM From Scratch** | Hand-built transformer — BPE tokenization, multi-head self-attention, full train/eval loop with checkpointing | PyTorch |
| **Medical RAG Chatbot** | Context-bound medical QA with strict hallucination-minimizing prompting and an AWS-ready containerized architecture | LangChain, Pinecone, Flask, Docker |
| **AI Travel Recommendation Platform** | Constraint-aware itinerary generation over multi-city PDF corpora with automated PDF export | LangChain, FAISS/ChromaDB, OpenAI embeddings |
